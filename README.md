<section>
  <header>
    <h1>Learning Station Designer</h1>
    <p>
      A digital tool for designing, planning, and structuring learning stations.
      Built for educators, trainers, instructional designers, and learning experience creators,
      it helps turn ideas into clear, engaging, and well-organized learning experiences.
    </p>
  </header>

  <section>
    <h2>Overview</h2>
    <p>
      <strong>Learning Station Designer</strong> is created to make the process of building learning stations
      more structured, flexible, and practical. It supports users in defining learning goals,
      organizing station modules, identifying expected outcomes, and selecting suitable content delivery methods.
    </p>
    <p>
      The tool can be used for in-person, online, and hybrid learning environments,
      making it suitable for everything from single activities to multi-step learning experiences.
    </p>
  </section>

  <section>
    <h2>Why Use Learning Station Designer?</h2>
    <ul>
      <li>Simplifies the learning station design process.</li>
      <li>Helps align goals, outcomes, activities, and assessment.</li>
      <li>Supports modular design for flexible learning experiences.</li>
      <li>Encourages active and learner-centered engagement.</li>
      <li>Makes planning more visible, editable, and shareable.</li>
      <li>Supports different content formats and teaching strategies.</li>
    </ul>
  </section>

  <section>
    <h2>Who Is It For?</h2>
    <ul>
      <li>Teachers and instructors</li>
      <li>University faculty</li>
      <li>Instructional and learning experience designers</li>
      <li>Corporate training teams</li>
      <li>Workshop facilitators</li>
      <li>Anyone designing learner-centered educational experiences</li>
    </ul>
  </section>

  <section>
    <h2>Key Features</h2>
    <ul>
      <li>
        <strong>Modular station planning:</strong>
        Build learning stations as standalone units or as connected parts of a larger sequence.
      </li>
      <li>
        <strong>Learning goal definition:</strong>
        Set clear, focused, and measurable goals for each station.
      </li>
      <li>
        <strong>Outcome mapping:</strong>
        Identify what learners should know, do, or demonstrate at the end of the experience.
      </li>
      <li>
        <strong>Content delivery options:</strong>
        Combine reading, video, discussion, hands-on tasks, case studies, collaboration, and more.
      </li>
      <li>
        <strong>Flow and timing support:</strong>
        Organize the sequence, duration, and logic of each module.
      </li>
      <li>
        <strong>Assessment integration:</strong>
        Add feedback and evaluation steps that match the purpose of each station.
      </li>
      <li>
        <strong>Flexible learning environments:</strong>
        Design for physical, digital, or hybrid learning spaces.
      </li>
    </ul>
  </section>

  <section>
    <h2>How It Works</h2>
    <ol>
      <li>Define the title and theme of your learning station.</li>
      <li>Identify the target audience and learning context.</li>
      <li>Write the learning goals.</li>
      <li>Create expected learning outcomes for each module.</li>
      <li>Select content formats and activity types.</li>
      <li>Plan the sequence, duration, and interaction flow.</li>
      <li>Add assessment and feedback elements.</li>
      <li>Review, share, and improve your design.</li>
    </ol>
  </section>

  <section>
    <h2>Use Cases</h2>
    <ul>
      <li>Designing active learning stations for classroom teaching</li>
      <li>Planning workshops and short learning programs</li>
      <li>Creating structured corporate training modules</li>
      <li>Building hybrid or online learning experiences</li>
      <li>Supporting team-based and project-based learning activities</li>
    </ul>
  </section>

  <section>
    <h2>Design Approach</h2>
    <p>
      Learning Station Designer treats learning experiences as flexible systems made up of connected modules
      rather than one fixed activity. This makes it easier to design each station with purpose,
      clarity, and alignment across goals, activities, and outcomes.
    </p>
    <p>
      The approach is grounded in active learning, learner-centered design, modular planning,
      and the thoughtful use of multiple content formats.
    </p>
  </section>
